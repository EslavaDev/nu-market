# Installed

## 1 step

```
yarn install
```
or
```
npm install
```
## 2 step

```
cd ios && pod install
```

## 3 step
```
yarn start --reset-cache
```
or
```
npm run start --reset-cache
```

## 4 step
```
yarn ios
```
or
```
npm run ios
```
